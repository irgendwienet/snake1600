# 1600 LED Vorhang

4 Stück Vorhang mit 20x20 LEDs = 1600 LEDs

## Strom


Je LED / Farbe < 13 mA

Aber es skaliert nicht wenn man alle LEDs anktiviert. Ein 400 LEDs (1 Vorhang) braucht bei voll weiß ca 4 A.

 - 100x blau = 1,24 A
 - 100x rot = 1,25 A
 - 100x grün = 1,23 A
 - 100x weiß (255/255/255) = 2,3 A
 - 200x rot = 2,3 A
 - 400x rot = 3 A
 - 400x weiß = 3,4 A
